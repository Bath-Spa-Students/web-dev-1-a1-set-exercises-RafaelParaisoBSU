HTML and CSS Reference Guide

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure and content of a webpage by using a system of tags and attributes.

HTML Page Structure

                <!DOCTYPE html> 

                <html lang="en"> 

                <head> 

                    <meta charset="UTF-8"> 

                    <meta name="viewport" content="width=device-width, initial-scale=1.0"> 

                    <title>Document</title> 
 
                </head> 

                <body> 

                    
                </body> 
 
                </html>

Paragraphs

<p> (short for "paragraph") is a markup tag used to define a paragraph of text. It is one of the fundamental block-level elements in HTML used for structuring textual content. Paragraphs are typically used to group together sentences or related content to form coherent blocks of text. 

                <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. </p>

Line Break

The <br> tag in HTML stands for "line break." It is a self-closing tag used to create a line break within text content. When the browser encounters a <br> tag, it inserts a line break, causing the text following the tag to start on a new line. 

                Lorem ipsum dolor sit amet consectetur adipisicing elit. <br>
                New line starts here!

Div Tag

The div tag, <div>, is an HTML element used to create a division or a section in a webpage. It's a generic container that allows you to group together other HTML elements and apply styles or scripts to them as a unit. 

                <div id="header">
                <h1>Welcome to my website</h1>
                <p>This is a paragraph of text.</p>
                <img src="example.jpg" alt="Example Image">
                </div>

Anchor Tag

<a> tag is an HTML element used to create hyperlinks within a webpage. It allows you to link to other web pages, files, locations within the same page, or even to different parts of the same document. 

                <a> href="https://www.example.com">Visit Example Website</a>

Image Tag

The image tag, <img>, is an HTML element used to embed images into a webpage. It allows you to display images from various sources, including local files or remote URLs. 

                <img src="example.jpg" alt="Example Image">

Span

<span>, is an HTML element used to apply styles or scripts to a specific section of text within a larger block of content. Unlike the div tag, which creates a block-level element, the span tag is an inline element, meaning it does not create a line break before or after the element. 

                    <span>Text to be styled or scripted</span>

HTML Lists

Unordered Lists <ul>, is an HTML element used to create an unordered list. This displays a bullet before each entry on the list. The list items must use the <li> tag. 

                    <ul>
                        <li>item 1</li>
                        <li>item 2</li>
                        <li>item 3</li>
                    </ul>

Ordered Lists <ol>, is an HTML element used to create an ordered list. This displays a numbering or numeric value before each entry on the list. The list items must use the <li> tag. 

                    <ol>
                        <li>item 1</li>
                        <li>item 2</li>
                        <li>item 3</li>
                    </ol>

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS separates the content of a webpage from its design, allowing developers to control the layout, colors, fonts, and other visual aspects of a website.

CSS Structure

                   .body {
                        margin: 0;
                        padding:  0;
                   }

Applying CSS

Inline CSS

Inline CSS refers to the practice of applying CSS styles directly within HTML elements using the style attribute. Instead of defining styles in an external CSS file or within a <style> tag in the document's <head> section, inline CSS attaches styles directly to individual HTML elements. 

                 <p style="color: red; font-size: 18px;">This is a paragraph with inline CSS.</p>

Internal CSS

Internal CSS, also known as embedded CSS or internal stylesheet, involves placing CSS rules directly within the <style> element in the <head> section of an HTML document. 

                <head>
                <style>
                    body {
                        font-family: Arial, sans-serif;
                        background-color: #f0f0f0;
                    }
                </style>
                </head>

External CSS

External CSS involves defining CSS rules in a separate file with a .css extension and then linking that file to an HTML document using the <link> element in the <head> section. This approach separates the styling from the HTML content, making it easier to maintain and update styles across multiple web pages. 

                in index.html:
                <link rel="stylesheet" href="style.css" class="stylesheet">

                in style.css:
                body {
                    font-family: Arial, sans-serif;
                    background-color: #f0f0f0;
                }

CSS Selectors

CSS selectors are patterns used to select and style HTML elements. They define which elements in an HTML document will be affected by the CSS rules.

Element Selectors

Selects HTML elements based on their names. (body, h2, a, p)

                    h1 {
                        color: blue;
                    }

ID Selectors

The ID selector in CSS is used to target a specific HTML element based on its unique ID attribute. An ID selector is prefixed with a hash symbol (#) followed by the ID value of the element.

                    #container {
                        position: absolute;
                        margin: 0;
                    }

CSS Properties

CCSS properties are attributes that define various stylistic aspects of HTML elements. These properties control how elements are displayed on a web page. Here are some examples of CSS Properties.

Color Properties (Color)

Changes the color of the text.

                    color: #ff0000;

Margin Property (Layouts)

Changes the margin between 2 elements.

                    margin: 10px;

Border Property (Box Model)

Creates a border around the box model.

                    border: 1px solid #000000;

Font-size Property (Typography)

Changes the size of the text.

                    font-size: 16px;

Background-Color Property (Background)

Sets the background color to a specified color.

                    background-color: #f0f0f0;
